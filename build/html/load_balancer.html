load_balancer module¶

class load_balancer.LoadBalancer(ip, port, num_servers, algorithm='random')[source]¶

Bases: object

Socket implementation of a load balancer. For the first time, the client connects to the load balancer. The load balancer assigns a server to the client, and from then on the client directly communicates with the server
